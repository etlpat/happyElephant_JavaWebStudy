// 本模块用于实现客户端为数据库增加用户信息

// 客户端首页index.jsp，调用dao输出数据库中全部用户的信息
// 首页可以选择进入insertUser界面，输入要添加的用户信息
// 输入信息后，若输入成功则重返首页，若输入失败则重新输入。


// src目录介绍：

// bean目录，用于存放基本java类
// ①RequestParamUser用于存放请求中的用户信息
// ②User用于存放完整的用户信息

// util目录，用于存放jdbc的工具类
// 该类：方法①实现加载驱动、连接数据库  方法②关闭资源

// dao目录，用于存放jdbc的CRUD方法
// 该类：方法①实现向数据库添加元素     方法②使用集合获取全部行

// servlet目录，用于存放Servlet类
// 本模块，只有insertUser页面需要用到servlet


// web目录介绍：

// index：用于存放首页（打印用户表、跳转到增加用户界面）

// insertUser：用于实现增加用户信息

// insertSuccess：若增加成功，则跳转到该页面
